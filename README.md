# MAX Channel Publisher Bot

Бот для мессенджера [MAX](https://max.ru), который публикует сообщения из личной переписки в указанный канал. Отправьте боту текст или картинку — он опубликует пост в канал с кнопкой-ссылкой.

## Возможности

- Публикация текстовых постов в канал через личное сообщение боту
- Публикация постов с картинками
- Автоматическое добавление кнопки-ссылки к каждому посту
- Ограничение доступа по списку администраторов (поддержка нескольких админов)
- Защита от дублирования: бот игнорирует сообщения из самого канала
- Готов к расширению на видео и другие типы вложений

## Быстрый старт

### 1. Получите токен бота

Откройте диалог с [PrimeBot](https://max.ru/primebot) в мессенджере MAX, следуйте инструкциям и создайте нового бота. После создания PrimeBot отправит вам токен.

### 2. Настройте окружение

```bash
cp .env.example .env
```

Заполните `.env`:

| Переменная | Описание | Пример |
|---|---|---|
| `BOT_TOKEN` | Токен бота от PrimeBot | `abc123...` |
| `CHANNEL_CHAT_ID` | ID канала для публикации | `-1001234567890` |
| `BUTTON_TEXT` | Текст кнопки на посте | `Написать боту` |
| `BUTTON_URL` | Ссылка кнопки | `https://max.ru/u/your_bot` |
| `ADMIN_USER_ID` | ID администраторов (через запятую) | `5386877,1234567` |
| `NODE_ENV` | Окружение | `production` |

> Если `ADMIN_USER_ID` не задан, публиковать сможет любой пользователь.

### 3. Запуск

#### Локально

```bash
npm install
npm start
```

#### Docker

```bash
docker compose up -d
```


## Как это работает

1. Вы пишете боту в личку
2. Бот проверяет, есть ли у вас права администратора
3. Извлекает текст и вложения из сообщения
4. Публикует пост в канал с кнопкой-ссылкой
5. Отправляет вам подтверждение
